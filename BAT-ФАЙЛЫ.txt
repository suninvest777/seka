🎯 START.bat ⭐ БЫСТРЫЙ ЗАПУСК
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   ЧТО ДЕЛАЕТ:
   • Останавливает старые процессы
   • Запускает сервер (порт 3006)
   • Запускает клиент React (порт 3000)
   • Открывает приложение в браузере
   
   КОГДА ИСПОЛЬЗОВАТЬ:
   • Ежедневный запуск игры
   • Все зависимости уже установлены
   
   ⏱️ ВРЕМЯ ЗАПУСКА: 15-20 секунд

🎯 FULL-START.bat ⭐ ПЕРВЫЙ ЗАПУСК
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   ЧТО ДЕЛАЕТ:
   • Проверяет наличие зависимостей (node_modules)
   • Автоматически устанавливает недостающие зависимости
   • Запускает сервер (порт 3006)
   • Запускает клиент React (порт 3000)
   • Открывает приложение в браузере
   
   КОГДА ИСПОЛЬЗОВАТЬ:
   • Первый запуск проекта
   • После клонирования проекта
   • После обновления зависимостей
   
   ⏱️ ВРЕМЯ ЗАПУСКА: 2-5 минут (первый раз), затем 30-60 секунд

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💡 РЕКОМЕНДАЦИИ ПО ИСПОЛЬЗОВАНИЮ

   1️⃣ ПЕРВЫЙ ЗАПУСК:
      → Используйте: FULL-START.bat

   2️⃣ ЕЖЕДНЕВНАЯ ИГРА:
      → Используйте: START.bat (быстро и просто!)

   3️⃣ ЗАВЕРШЕНИЕ РАБОТЫ:
      → Всегда используйте: STOP.bat
      → Перед выключением компьютера
      → Перед запуском другого проекта

   4️⃣ РЕШЕНИЕ ПРОБЛЕМ:
      → Запустите STOP.bat
      → Подождите 5 секунд
      → Запустите FULL-START.bat заново
