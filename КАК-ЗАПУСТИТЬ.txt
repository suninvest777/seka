📋 ЧТО ТРЕБУЕТСЯ:
   • Node.js (версия 14 или выше)
   • npm (устанавливается вместе с Node.js)
   • Python 3.x (опционально, для HTTP сервера)

🚀 КАК ЗАПУСТИТЬ:

   1️⃣ ПЕРВЫЙ ЗАПУСК:
      Дважды кликните на файл: FULL-START.bat
      
      ⚠️ Первый раз может занять 2-5 минут (установка зависимостей)
      ✅ Последующие запуски будут намного быстрее

   2️⃣ ЕЖЕДНЕВНАЯ ИГРА:
      Дважды кликните на файл: START.bat ⭐
      
      ⚠️ Быстрый запуск (~15 секунд)
      ✅ Если зависимости уже установлены

   3️⃣ ОСТАНОВКА:
      Дважды кликните на файл: STOP.bat

📁 ПОСЛЕ ЗАПУСКА БУДУТ ОТКРЫТЫ 2 ОКНА:
   1. 🎮 Сервер игры (порт 3006) - НЕ ЗАКРЫВАТЬ!
   2. 🌐 Клиент React (порт 3000) - НЕ ЗАКРЫВАТЬ!

🌐 ДОСТУП К ИГРЕ:
   • С ПК: http://localhost:3000
   • С телефона: http://[IP-вашего-PC]:3000
   • API сервера: http://localhost:3006/api

⚠️ ВАЖНО:
   • НЕ ЗАКРЫВАЙТЕ окна командной строки во время игры
   • Если страница не открывается, подождите 10-15 секунд
   • Проверьте, что в окне сервера написано: "🎮 Сервер запущен на порту 3006"
   • Убедитесь, что порты 3000 и 3006 не заняты другими программами

🛑 КАК ОСТАНОВИТЬ:
   Дважды кликните на: STOP.bat

🆘 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ:
   1. Убедитесь, что Node.js установлен: node --version
   2. Закройте все окна и запустите STOP.bat
   3. Перезапустите FULL-START.bat
   4. Подождите 30-60 секунд для полной загрузки
   5. Проверьте, что порт 3000 не занят VPN
